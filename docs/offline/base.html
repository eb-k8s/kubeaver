概述 ​

k8s基础包主要包括k8s组件（coredns、etcd、kube-apiserver、kube-controller-manager、kube-proxy和kube-scheduler）、网络插件flannelv0.22.0和metrics-server。涉及到的工具软件有net-tools、nmap、tcpdump、sysstat和iotop。
