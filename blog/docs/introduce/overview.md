# 概述

Kubeaver 是一款开源的，专为完全离线环境设计的Kubernetes 图形化部署与管理平台。它主要面向系统管理员，提供了一套完整、灵活、高效的解决方案，支持在完全无外网连接的环境中，快速、便捷地完成Kubernetes集群的离线部署与全生命周期管理，同时还支持系统应用和第三方应用的全离线部署，大幅降低了在严格隔离场景下的技术门槛与运维成本。

目前kubeaver已获得云原生计算基金会（CNCF）的官方认可，正式成为 [CNCF 认证的Kubernetes Installer 项目](https://www.cncf.io/training/certification/software-conformance/)
